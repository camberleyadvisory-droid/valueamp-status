# ValueAmp Status

Public uptime monitor and status page for the
[ValueAmp platform](https://app.valueamp.co.uk), powered by
[Upptime](https://upptime.js.org). GitHub Actions check the site every 5 minutes
and publish the results here and to **https://status.valueamp.co.uk**.

This repository contains **no application code** — only uptime configuration and
the recorded availability history.
